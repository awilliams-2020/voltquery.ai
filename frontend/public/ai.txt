# AI Agent Discovery Metadata for Volt Query AI

## About
Volt Query AI is an AI-powered RAG (Retrieval-Augmented Generation) SaaS application that provides intelligent insights about EV charging stations, electricity rates, solar energy production, and energy system optimization.

## Agent Information
- Name: Volt Query AI
- Version: 1.0.0
- Type: Web Application / RAG System
- Domain: EV Infrastructure, Energy Analytics, Renewable Energy
- Language: English (en-US)

## Capabilities
- EV charging station location queries
- Electricity rate and utility cost queries
- Solar energy production estimates
- Energy system optimization analysis
- Natural language question answering
- Location-based data retrieval

## Data Sources
- NREL Alternative Fuels Data Center API (EV charging stations)
- NREL PVWatts API (solar production estimates)
- OpenEI URDB API (utility rates)
- NREL REopt API (energy optimization)

## API Endpoints
- POST /api/rag/query - Natural language query endpoint
- POST /api/fetch-stations - EV charging station data
- GET /api/history/queries - Query history
- GET /api/history/stats - Usage statistics

## Authentication
- Required: Yes (via Clerk authentication)
